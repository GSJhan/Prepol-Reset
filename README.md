# 🐕 PREPOL RESET - Juego Cívico Educativo

Un juego web interactivo para aprender cívica peruana de forma divertida.

## 🎮 Características :v

- 🏘️ 4 Distritos (Barrio, Distrito, Región, País)
- 📚 60+ Preguntas sobre cívica peruana
- 🎯 Sistema de Soles Cívicos
- 🏆 Rangos progresivos
- ❤️ Sistema de vidas
- 🥇 Leaderboard global
- ⚔️ Duelos multijugador (próximamente)

## 🚀 Cómo Usar El Juego xd

1. Clona el repositorio
2. Abrir "index.html" en tu navegador
3. Y ¡A jugar!
